(1): By Definition 2.1.1,

\begin{align*} \ip{x}{y+z} \amp = \cc{\ip{y+z}{x}}\\ \amp = \cc{(\ip{y}{x} + \ip{z}{x})}\\ \amp = \cc{\ip{y}{x}} + \cc{\ip{z}{x}}\\ \amp = \ip{x}{y} + \ip{x}{z} \end{align*}

Parts 2, 3 are left as an exercise.

(4): If \(\ip{x}{z} = \ip{y}{z}\text{,}\) then

\begin{align*} 0 \amp = \ip{x}{z} - \ip{y}{z}\\ \amp = \ip{x}{z} + \ip{-y}{z}\\ \amp = \ip{x-y}{z}. \end{align*}

Assuming that this statement holds for all \(z \in V\) means that it holds for \(z = x-y\text{.}\) But then \(\ip{x-y}{x-y} = 0\text{,}\) so by Definition 2.1.1 (4), it must be that \(x - y = 0\) and so \(x = y\text{.}\)
