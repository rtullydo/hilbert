Let \(X\) be a set. A metric is a map \(d:X\times X \to \R\) which satisfies for all \(x, y, z \in X\)

  1. \(d(x,y) \geq 0\text{;}\)
  2. \(d(x,y) = 0\) if and only if \(x = y\text{;}\)
  3. \(d(x,y) = d(y,x)\text{;}\)
  4. \(d(x,z) \leq d(x,y) + d(y,z)\) for all \(y\text{.}\)
